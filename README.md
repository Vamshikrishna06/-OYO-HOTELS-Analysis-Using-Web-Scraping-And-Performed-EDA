# OYO Hotel Data Web Scraping Project

## Overview

This project focuses on **web scraping OYO hotel listing data** to collect information such as hotel names, locations, prices, ratings, and available facilities. The extracted data is stored in a structured format for further analysis, visualization, and data-driven insights.

## Features

* Web scraping of hotel listings
* Extraction of hotel details such as name, price, rating, and location
* Data cleaning and preprocessing
* Storage of scraped data in CSV/Excel format
* Ready-to-use dataset for Exploratory Data Analysis (EDA) and visualization

## Technologies Used

* **Programming Language:** Python
* **Libraries:** BeautifulSoup, Requests, Pandas
* **Data Storage:** CSV / Excel

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/oyo-web-scraping.git
   ```
2. Navigate to the project folder:

   ```bash
   cd oyo-web-scraping
   ```
3. Install required libraries:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the scraping script to collect hotel data:

```bash
python scraper.py
```

The scraped dataset will be saved automatically in the project directory.

## Applications

* Hotel price trend analysis
* Location-based hotel comparison
* Data visualization and dashboard creation
* Machine learning dataset preparation
